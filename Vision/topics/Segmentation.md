**k-Means**
In noisy image, threshold values are harder to identify
![[Pasted image 20260531114509.png|394]]
![[Pasted image 20260531114520.png|398]]
algorithm
- doesn't require pre-labelling
- goal
	- choose 3 centres as representative intensities
	- label for every pixel which centre it's nearest to
	- Best centers are those that 
		- minimise SSD (squared sum distances)
		- each distance is between a point and its nearest cluster center
- Intuition
	- If we knew centres we can easily allocate points to centres
	- if we knew group memberships we could easily get centres by averaging point positions
	- Start random and iterate
- Method
	- Initialise k cluster centers
		- c1, ..., ck
	- Given centres determine points in each cluster
	- Set centres to be mean of points in cluster
	- repeat
- Properties
	- Converges to some solution
	- Can be local minimum (not best solution)
	- Simple, fast to compute
	- Need to set k
	- Sensitive to initial centres
	- Sensitive to outliers
	- Detects spherical clusters only
		- ![[Pasted image 20260531115234.png|248]]

**Probabilistic clustering**
We need to be able to handle non-circular, potentially overlapping clusters
Gaussian Mixture models
- Clusters modelled as Gaussians not just mean
- EM algorithm: assign data to cluster with some probability
- Gaussian distribution
	- 1D (univariate) describes one variable
		- parameters $\mu, \sigma$
	- Multivariate
		- vector containing mean position $\mu$
		- symmetric "positive definitite" covariance matrix $\Sigma$
			- describes shape of gaussian
			- ![[Pasted image 20260531123948.png|540]]
			- revisited for local features
Method
- Treat data as being generated by sampling a continuous function
	- function called generative model
	- can identify outliers (>3 standard deviations)

**Expectation Maximisation**
Goal
- Find blob parameters that maximise likelihood function
	- $P(data|\theta)=\prod_x P(x|\theta)$
Approach
- E-step
	- Given current guess of blobs compute ownership of each point
- M-step
	- Given ownership probabilities, update blobs to maximise function
- Repeat until convergence
