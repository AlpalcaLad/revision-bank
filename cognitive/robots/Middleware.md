**ROS (ROBOT OPERATING SYSTEM)**
Open source flexible middleware designed for building complex robotic applications rather than a traditional OS. Built ontop of linux.
Nodes publish on topics that are read by subscriber nodes
used by TurtleBot 4

**YARP (Yet another robot platform)**
Used for iCub, smaller community.
Same publisher/subscriber method
Mainly c++

**NAOQI**
proprietary software by softbank robotics
used for nao pepper and romeo
